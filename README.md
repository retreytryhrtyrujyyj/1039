# 旅游网站

## 项目简介
旅游网站是基于Java语言，SSM框架，Mysql数据库的旅游网站系统。包含前台系统和后台展示系统。分用首页展示，户模块，游玩路线,游玩攻略，在线预订。后台管理系统分为：用户管理、酒店管理，预订管理，景区管理等模块。适合毕业设计、大作业、课程设计。

2. 开题报告（任务书）+ 配套论文（一万两千字以上）+ 答辩PPT
3. 远程控屏调试（向日葵或者toDesk控屏）
4. 一键启动项目（双击运行演示，无需搭建复杂环境）
